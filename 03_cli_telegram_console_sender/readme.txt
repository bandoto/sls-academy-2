https://t.me/ConsoleSenderCLIBot

INSTALL PACKAGES => npm i

RUN APP => node index.js s|start (run to get current chat id, before run enter /start in bot)
           node index.js m|message "type message" (run to send message)
           node index.js p|photo "type path" (run to send photo)
           
BUILD JAVASCRIPT FILE => npm run build
